This plugin adds the functionality to take various online payments on your store using Zeus for WooCommerce.

== Description ==

The Payment Gateway Zeus for WooCommerce plugin adds the functionality to take Zeus payments on your store using Zeus for WooCommerce.

= About Zeus =

Zeus is a comprehensive payment platform that offers various online payment methods, such as credit card payment and convenience store payment in Japan.

[Zeus](https://www.cardservice.co.jp/)

In order to use Zeus, you have to purchase the authentication key at the following site.

[WordPress Market](https://www.wpmarket.jp/)

== Installation ==

1. Upload the plugin file in the plugin install page or copy the `wc-zeus-gateway` directory into your `wp-content/plugins` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. That's it! :)
